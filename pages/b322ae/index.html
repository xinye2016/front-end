Canvas入门

# Canvas

# 1. Canvas 基础

# 1. 什么是 Canvas

就是 HTML 5 给出的一个可以展示绘图内容的标签，最早是苹果公司提出的该标签。

# 2. 基本使用

  1. 提供 Canvas 标签即可， 默认就会占据 300 * 150 的区域，利用 html 属性为它设置宽高，不要使用 CSS 来设置。
<canvas id="tutorial" width="150" height="150"></canvas>
1

⚠️ 注意：

  • 使用属性设置 canvas 标签的宽高, 实际上相当于增加了 canvas 画布的像素
  • 如果使用 CSS 来设置画布的大小, 那么不会增加像素点, 只是将像素扩大了
  1. Canvas 只能展示绘图的内容. 但是不能进行绘图，利用 Canvas 找到绘图工具，每一个 Canvas 都有一套工具, 利用工具可以在当前 Canvas 上进行绘图，使用语法 canvas.getContext( '2d' ) 就返回一个在当前画布上绘图的工具集，这个工具集专门绘制平面图形，如果要绘制立体的图形需要传入'webgl'，绘制平面图形的对象是 CanvasRenderingContext2D 类型的。
var canvas = document.getElementById('tutorial');
var ctx = canvas.getContext('2d');
1
2
  1. 开始绘图，首先需要绘制点， 然后需要将这些点描边才可以看到东西。

# 3. 常用的绘图方法

  • beginPath() 新建一条路径，生成之后，图形绘制命令被指向到路径上生成路径。
  • closePath() 闭合路径之后图形绘制命令又重新指向到上下文中
  • ctx.moveTo( x, y ) 将绘图的起始点设置为 x,y。
  • ctx.lineTo( x, y ) 从当前位置, 绘制直线到 x,y。
  • ctx.stroke() 就是将刚刚绘制的所有的点联系起来. 就可以看到图形了。
  • ctx.fill() 调用 fill 方法会将所有的点连接起来， 并构成一个封闭的图形结构，如果所有的描点没有构成封闭结构，也会将开始的起点, 与最终的点连接起来,构成一个闭合的图形, 并在里面填充颜色(默认黑色)。
